# Portfolio

This repository contains the source code for my personal portfolio website.
Overview

The site is designed with a retro-cyber aesthetic, reflecting my focus on game design and development. It serves as a central hub for my projects, character concepts, and visual archives.

## **Site Structure**

- **Hero & About:** Introduction, technical skills, tools, and personal inventory.
- **Projects:** Detailed logs such of 2D platformers, 3D concepts, and papercraft prototypes.
- **Characters:** Showcase of character designs, including roles, stats, and lore.
- **Archive:** Visual impressions presented in a polaroid style gallery.
- **Restricted Area:** Protected section for game design documentation (unlockable via passcode).
- **Contact:** Direct connection channels.

## **Key Features**

- **Bilingual Support:** Instant toggle between English and German.
- **Audio Player:** Integrated custom audio controls.
- **Interactive Elements:** Draggable windows and dynamic stickers.
- **Responsive Design:** Fully adaptable layout for desktop and mobile devices.
